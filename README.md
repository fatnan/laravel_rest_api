## How To Install
- composer install
- composer require laravel/sanctum
- php artisan vendor:publish --provider="Laravel\Sanctum\SanctumServiceProvider"
- cp .env.example .env
- setting your .env like DB_DATABASE,DB_HOST and DB_PASSWORD
- php artisan migrate
